3^{8}\equiv1515\equiv5\bmod23\
3^{16}\equiv55\equiv3\bmod23\

所以3^{2019}\bmod23=9

9.编写Python语言程序完成欧拉Phi函数的计算，即输入正整数n，计算并返回φ(n).

import math

def gcd(a,b):
    if b==0:
        return a
    else:
        return gcd(b,a%b)
    
def jud(num):#判断素数
    a=1
    for i in range(2,math.ceil(math.sqrt(num))):
        if num%i==0:
            a=0
    if a==1:
        return 1
    else:
        return 0

def Phi(n):
    if jud(n)==1:
        return n-1
    else:
        count=0
        for i in range(1,n):
            if gcd(n,i)==1:
                count=count+1
        return count

10.设p是素数，计算(p-1)! mod p,并找出规律，写成定理，并给出证明。

证明：
$$
因为p是一个素数，所以小于p的所有正整数都与p互素，\
对于1\leq c< p,总存在c的唯一的确定的乘法逆元,\
使得cc^{-1}\equiv1\bmod p(c^{-1}<p),\
即c^{-1}存在与{1.2.3…p-1}的最小剩余系,\
除1和p-1外，存在i,j\in{2.3.4…p-2}\
使得ij\equiv 1\bmod p,\
所以(p-1)!\equiv p-1\bmod p,\
所以(p-1)!\bmod p =p-1 成立。\
$$
