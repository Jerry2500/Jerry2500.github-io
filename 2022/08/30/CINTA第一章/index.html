  1. 用 C 语言编程实现一种迭代版本的简单乘法。
int multiple(int a,int b)
{
    if(b==0) return 0;
    int res=0;
    while(b)
    {
        if(b&1) res+=a;
        a<<=1;
        b>>=1;
    }
    return res;
}

  1. 给定一个整数 v，如何判断 v 是否 2 的某次方？比如，v = 4 = 22，返回 T rue； v = 9 = 23 + 1 并非 2 的次方，返回 False。请写一个 C 语言的函数来实现以上功 能。
int f(int v)
{
    if(v==1) return true;
    else if(v==0) return false;
    else if(v&1) return false;
    return f(v>>1);
}
